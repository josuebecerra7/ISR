## Installation
#### Prerequisites
 - [Ubuntu 20.04.2 LTS](https://releases.ubuntu.com/20.04/)
 - [ROS Noetic](http://wiki.ros.org/noetic/Installation/Ubuntu)

## Usage: 
 1. Go into catkin_ws
 2. Source the project:
    
```
source devel/setup.bash
```
 3. Start the ROS core:
 ```
roscore
```
